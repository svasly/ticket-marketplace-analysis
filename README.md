# Анализ лояльности пользователей Яндекс Афиши

## Описание проекта

Проект посвящён исследованию факторов, влияющих на повторные покупки пользователей билетного сервиса Яндекс Афиша. Цель — выявить характеристики пользователей и их первых заказов, которые связаны с вероятностью возврата на платформу и совершения повторных заказов.


Результаты помогут отделу маркетинга оптимизировать стратегии привлечения и удержания клиентов.


## Структура проекта

- **Загрузка и предобработка данных** – подключение к базе данных PostgreSQL, выгрузка, очистка, приведение валют, фильтрация выбросов.
- **Построение профиля пользователя** – агрегация признаков по каждому пользователю, создание бинарных меток `is\_two` (2+ заказа) и `is\_five` (5+ заказов).

- **Исследовательский анализ данных (EDA)**:

  - Влияние признаков первого заказа (тип мероприятия, устройство, регион, билетный оператор) на возврат.
  - Связь выручки и количества билетов с лояльностью.

  - Анализ временных характеристик (день недели, интервалы между заказами).

  - Корреляционный анализ с использованием коэффициента `phi\_k`.

- **Выводы и рекомендации** – практические предложения для маркетинга.


## Используемые данные


Исходные данные получены из базы данных Яндекс Афиши (таблицы `purchases`, `events`, `city`, `regions`). Дополнительно использован датасет с курсами казахстанского тенге к рублю за 2024 год.


**Основные поля после выгрузки:**

- `user\_id` – идентификатор пользователя

- `order\_dt` – дата заказа

- `revenue` – выручка в исходной валюте

- `tickets\_count` – количество билетов

- `event\_type\_main` – тип мероприятия

- `device\_type\_canonical` – тип устройства (mobile/desktop)

- `region\_name`, `city\_name` – место проведения

- `service\_name` – билетный оператор

## Ключевые результаты

* `61.7%` пользователей совершают повторные покупки (2+ заказа). `29.1%` – лояльные (5+ заказов).
* Лучшие точки входа по возвратам: театр `(63.6%)` и концерты `(61.7%)`, худшие – спорт `(56.1%)` и ёлки `(54.7%)`.
* Пользователи с десктопа возвращаются чаще, чем с мобильных устройств (`63.6% `против `60.9%`).
* Самый «лояльный» сегмент по количеству билетов – `2–3 билета` (доля возвратов `73.3%`, `43.7%` всех пользователей).
* Золотое окно удержания – `1–14` дней после предыдущего заказа. После `14` дней вероятность повторной покупки резко падает.
* Корреляционный анализ показал, что характеристики первого заказа (тип события, устройство, оператор) слабо связаны с итоговым числом заказов. Ключевые драйверы лояльности – `lifetime_days` (время жизни клиента) и `avg_tickets_count` (среднее число билетов в заказе).

## Рекомендации для маркетинга

1. Стимулировать повторную покупку в первые `14` дней – запустить триггерные коммуникации на `7-й` и `12-й` день после заказа.
2. Акцентировать акции на покупку `2–3` билетов – это самый доходный и лояльный сегмент.
3. Улучшить мобильный опыт – пользователи мобильных устройств возвращаются реже.
4. Оптимизировать контентную стратегию – делать упор на театр и концерты, для спорта и ёлок разработать дополнительные удерживающие механики.
5. Не закрывать «слабые» каналы привлечения – лояльность не зависит от канала первого касания, поэтому можно продолжать привлекать трафик из разных источников, фокусируясь на пост-продажном сервисе.
